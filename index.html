    I developed an AI-based monitoring system called BrainyBits using Python, Flask, Django, and SQL to track and analyze system performance and anomalies. This system automated 50% of monitoring tasks, significantly improving efficiency, and reduced reporting time by 25% through effective data analysis and visualization using Jupyter Notebook and Excel. (August ’24)
